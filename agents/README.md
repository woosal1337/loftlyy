# Translation Agents

These agent prompts are lightweight operating guides for expanding the site's translations.

- `translation-scaffolder.md`: prepares a new locale file and identifies config changes.
- `translation-translator.md`: translates a batch of strings while preserving placeholders and brand names.
- `translation-reviewer.md`: checks key parity, placeholder safety, and tone consistency.

Useful commands:

```bash
pnpm translations:status
pnpm translations:scaffold -- it
pnpm validate
```
